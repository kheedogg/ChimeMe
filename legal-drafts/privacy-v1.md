## 1. 수집하는 개인정보 항목

### 1.1 회원 가입 시 (필수)
- **전화번호** (Firebase Phone Auth 인증용)
- **닉네임** (그룹 내 표시명)
- **생년월일** (만 14세 이상 가입 확인용 — Q-12-03)
- **약관 동의 이력** (이용약관, 개인정보처리방침)

### 1.2 회원 가입 시 (선택)
- **프로필 사진**
- **마케팅 정보 수신 동의 여부**

### 1.3 서비스 이용 시 자동 수집
- **3초 영상 (콘텐츠)** — 그룹 내 공유 목적
- **채팅 메시지** — 그룹 내 대화 목적
- **그룹 멤버십 정보** — 가입 그룹, 가입 일시
- **기기 정보** — OS 버전, 앱 버전, 기기 모델 (서비스 안정성 모니터링)
- **FCM 토큰** — 푸시 알림 발송 목적
- **시간대 (Timezone)** — 슬롯 표시 현지화 목적
- **신고 / 차단 이력** — 서비스 안전성 유지 목적

### 1.4 자동으로 생성·수집되는 정보
- 접속 로그 (IP 주소, 접속 시각)
- 서비스 이용 기록
- 오류 로그 (Sentry / Crashlytics — 익명화)

## 4. 개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다음의 경우에 한해 예외적으로 제공합니다.

1. 이용자가 사전 동의한 경우
2. 법령 규정 또는 수사기관의 적법한 절차에 따른 요청이 있는 경우
3. 통계 작성, 학술 연구 등 목적으로 특정 개인을 식별할 수 없는 형태로 제공하는 경우

## 5. 개인정보의 처리 위탁

회사는 서비스 운영을 위해 다음 업체에 일부 업무를 위탁합니다.

| 수탁업체 | 위탁 업무 | 처리 항목 |
|---|---|---|
| Google LLC (Firebase) | 인증, 데이터 저장, 클라우드 인프라 | 전화번호, 영상, 채팅, FCM 토큰 |
| Apple Inc. | iOS 푸시 알림 (APNs) | FCM 토큰 |
| Sentry (Functional Software, Inc.) | 오류 로그 수집 | 익명화된 오류 정보 |

### 5.1 국외 이전
- Firebase 데이터는 **서울 리전(asia-northeast3)**에 저장되나, 일부 메타데이터는 미국 서버를 경유할 수 있습니다.
- 본 이전에 동의하지 않으시는 경우 서비스 이용이 제한될 수 있습니다.

## 6. 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

1. **개인정보 열람 요구**
2. **개인정보 수정 / 삭제 요구**
3. **개인정보 처리 정지 요구**
4. **회원 탈퇴 및 데이터 삭제 요구** — 서비스 내 "계정 삭제" 메뉴
5. **개인정보 이동권** (GDPR 적용 시) — `support@chimeme.app`으로 요청

권리 행사는 서비스 내 설정 메뉴 또는 `privacy@chimeme.app` 이메일로 가능합니다.

## 7. 개인정보의 파기

### 7.1 파기 절차
- 회원 탈퇴 또는 보유 기간 경과 시 즉시 파기
- 자동 파이프라인 (Cloud Function)으로 안전하게 삭제

### 7.2 파기 방법
- **전자적 파일 형태**: 복구 불가능한 방법으로 영구 삭제
- **출력물 등 종이 형태**: 분쇄기로 파쇄 또는 소각

## 8. 개인정보의 안전성 확보 조치

회사는 다음의 기술적·관리적 보호 조치를 시행합니다.

1. **암호화 저장** — Firebase 자체 암호화 (AES-256) + HTTPS 전송
2. **App Check** — 비정상 클라이언트 차단 (D-12-04)
3. **접근 제어** — Firestore Security Rules 기반 권한 분리
4. **개인정보 처리자 교육** — 정기 보안 교육
5. **침해사고 대응** — 침해 발생 시 72시간 내 통지 (KISA / GDPR)
6. **물리적 접근 통제** — 사무실 출입 통제, 서버 데이터센터 접근 제한

## 9. 14세 미만 아동의 개인정보 보호

**ChimeMe는 만 14세 이상부터 이용 가능합니다.** (Q-12-03)
- 가입 시 생년월일 입력으로 연령 확인
- 만 14세 미만의 가입 시도는 차단
- 만 14세 미만이 부정 가입한 사실이 확인되면 즉시 계정 삭제

## 10. 자동 수집 장치 관리 (쿠키 / 토큰)

- ChimeMe는 모바일 앱으로 제공되며 일반 웹 쿠키는 사용하지 않습니다.
- 인증 토큰 (Firebase ID Token, Refresh Token)은 기기의 보안 저장소(`expo-secure-store`)에 안전하게 보관됩니다.
- 이용자는 앱 내 "로그아웃" 또는 "계정 삭제"를 통해 토큰을 폐기할 수 있습니다.

## 11. 푸시 알림

- 회사는 그룹 내 새 영상 / 채팅 메시지 / 시스템 알림을 위해 푸시 알림을 발송합니다.
- 이용자는 앱 내 "알림 설정"에서 알림 종류별 수신 여부를 변경할 수 있습니다.
- 시스템 알림 권한은 기기 설정에서 변경 가능합니다.
