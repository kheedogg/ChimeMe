# main — 메인 (그룹 리스트)

## 화면 키
`main`

## 진입로
- 인증된 사용자가 앱을 열면 첫 화면
- 다른 화면(그룹 상세, 채팅 등)에서 뒤로가기 → 이 화면

## 목적
- 사용자가 속한 그룹들을 한눈에 보고 활동 여부를 빠르게 파악
- 새 영상이 올라온 그룹에 즉시 진입할 수 있도록 유도
- 그룹 추가 진입점 제공

## 사용자 시나리오
1. **새 영상 확인**: 앱을 열면 "새 영상" 뱃지가 붙은 그룹부터 시선이 가도록 정렬. 카드 탭 → 그룹 상세에서 최신 시간대로 즉시 이동.
2. **현황 훑기**: 각 그룹의 멤버 수와 마지막 활동 시간을 한 줄로 확인.
3. **그룹 생성/초대**: 우측 + FAB로 그룹 생성 또는 초대 링크 진입.
4. **내 프로필**: 우상단 프로필 아이콘 탭.

## 성공 지표
- 진입 후 그룹 카드 첫 탭까지의 시간이 짧을 것
- 새 영상이 있는 그룹의 카드 시인성
