# blocked-users — 차단 사용자 목록

## 화면 키
`blocked-users`

## 진입로
- `profile` 화면 "차단한 사용자 (N명)" 메뉴

## 목적
- 본인이 차단한 사용자 목록 조회
- 개별 차단 해제

## 사용자 시나리오
1. 프로필 → 차단한 사용자 메뉴 진입
2. 차단한 사용자 리스트 확인
3. 항목 우측 "차단 해제" 탭 → 확인 다이얼로그 → 해제

## 정책 (Q-12-04)
- 차단 시: 차단자 화면에서 피차단자의 영상 / 메시지 / 프로필 즉시 가림
- 차단 해제 시: 가림 즉시 해제, 기존 콘텐츠 다시 표시
- 차단은 단방향 (B가 A를 차단해도 A는 B를 볼 수 있음. 단, A는 B 차단 사실을 모름)
- 차단된 사용자가 그룹 채팅에서 발송한 메시지는 차단자에게만 가려짐

## 차단 시점
- `chat` 메시지 길게 누르기 → "사용자 차단"
- `report` 화면에서 사용자 신고 시 자동 차단 (Q-12-04)
- `group-detail` 영상 길게 누르기 → "사용자 차단"

## 미정 / 향후
- 차단 사용자가 같은 그룹 멤버인 경우 그룹에서 자동 탈퇴 옵션 (현재는 가림만)
- 차단 시 이유 입력 (현재는 없음)
- 양방향 차단 (양쪽 모두 가림) — 향후 검토
