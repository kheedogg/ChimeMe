# capture — UI

## 레이아웃 (풀스크린 카메라)

### 1. 상단 영역 (반투명 오버레이)
- 좌측: ← X 닫기 (그룹 상세로 복귀)
- 가운데: 슬롯 정보
  - "**09:00 슬롯**" (큰 글씨)
  - "00:42:18 후 마감" (남은 시간 카운트다운, 옅은 그레이)
- 우측: 전·후면 카메라 토글 아이콘

### 2. 카메라 미리보기 (전체 영역)
- 화면 90% 차지
- 카메라 정사각형 또는 9:16 비율 (디바이스 화면에 맞춤)
- 미리보기 위에 상태별 오버레이:

#### 2.1 대기 상태
- 가이드 텍스트 (하단): "**셔터를 탭하면 3초가 시작돼요**"

#### 2.2 카운트다운 (3-2-1)
- 화면 가운데 큰 숫자 (`3` → `2` → `1`)
- 글자 크기 ~120px, 노란색 + 그림자
- 1초 간격, 스케일 애니메이션 (1.0 → 1.2 → 1.0)

#### 2.3 녹화 중 (3초)
- 화면 상단 빨간 점 + "REC" + 진행 바
- 진행 바: 좌→우 채워짐 (3초 = 100%)
- 진행 바 색상: 노란색 → 끝 직전 빨간색

#### 2.4 녹화 완료 → 미리보기
- 녹화된 영상 자동 재생 (음소거)
- 무한 루프
- 하단 좌우 액션:
  - **다시 찍기** (회색)
  - **올리기** (큰 노란 버튼)

### 3. 셔터 영역 (대기 상태에서만 표시)
- 화면 하단 가운데
- 큰 원형 셔터 버튼 (78×78px)
- 외곽 라인 + 내부 빨간 점 (대기 상태)
- 탭 시 카운트다운 시작

### 4. 권한 거부 화면 (카메라/마이크 권한 없을 때)
- 풀스크린 안내
- 일러스트 + 헤딩 "카메라 권한이 필요해요"
- 서브: "3초 영상을 찍으려면 카메라와 마이크 권한이 필요해요"
- 버튼: **설정 열기** (`Linking.openSettings()`)

### 5. 슬롯 마감 직전 경고
- 마감까지 5분 미만일 때:
  - 상단 카운트다운 빨간색
  - 하단에 펄스 애니메이션 안내: "곧 슬롯이 마감돼요"

### 6. 슬롯 마감 후
- 카메라 비활성 + 어두운 오버레이
- 가운데: "**이 시간 슬롯은 마감됐어요**"
- 버튼: "이전 슬롯 영상 보기" → 그룹 상세로

## 디자인 토큰
- `color.recordDot` — 빨강
- `font.countdown` — 굵은 sans-serif 120px
- `shutter.size` — 78
- `progress.barHeight` — 4

## 컴포넌트
- `CameraPreview` — 카메라 라이브 뷰 (`expo-camera`)
- `ShutterButton` — `{ onPress, disabled }`
- `Countdown` — `{ from=3, onComplete }`
- `RecordingProgress` — `{ duration=3000, onComplete }`
- `VideoPreview` — `{ uri, onRetake, onUpload }`
- `SlotInfoBar` — `{ slotLabel, remainingSeconds }`
- `PermissionDeniedScreen` — `{ onOpenSettings }`
- `SlotExpiredScreen` — `{ onGoBack }`

## 상태 머신
```
대기 → (셔터 탭) → 카운트다운 → 녹화 → 미리보기
                                       ↓ ↑
                                    다시 찍기 → 카운트다운 (재진입)
                                       ↓
                                    올리기 → 큐 등록 → 그룹 상세 복귀
```

## 인코딩 설정 (D-12-03, D-12-11 확정)
- 720p (1280×720)
- H.264 (`expo-camera` `recordAsync({ codec: 'avc1' })`)
- 30fps
- 비트레이트 2~3 Mbps
- MP4 컨테이너
- 클립 크기 약 1 MB

## 접근성
- 셔터 버튼 a11y: "촬영 시작, 한 번 탭하면 3초 카운트다운 후 자동 녹화"
- 카운트다운 a11y announce: "3", "2", "1"
- 녹화 완료 a11y: "녹화 완료, 미리보기 화면"
- 슬롯 마감 a11y: "이 시간 슬롯은 마감됐습니다"
