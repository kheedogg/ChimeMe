# profile — UI

## 레이아웃

### 1. 헤더
- 좌측 ← 뒤로가기 (메인으로)
- 가운데: **내 프로필**

### 2. 프로필 카드 (상단)
```
       ┌──────────┐
       │  사진    │  ← 탭 시 사진 변경
       │  편집 ✏ │
       └──────────┘
        [닉네임 ✏]
        +82 10-****-5678
```
- 원형 아바타 (110×110) 좌상단 작은 편집 아이콘
- 닉네임 옆 편집 아이콘 (탭 시 인라인 편집)
- 전화번호 read-only, 마스킹 표시

### 3. 메뉴 섹션 1: 설정

| 아이콘 | 라벨 | 우측 |
|---|---|---|
| 🔔 | 알림 설정 | > |
| 🚫 | 차단한 사용자 (N명) | > |
| 👥 | 내 그룹 (N개) | > |

### 4. 메뉴 섹션 2: 도움말 / 약관

| 아이콘 | 라벨 | 우측 |
|---|---|---|
| ❓ | 도움말 | > |
| 📄 | 이용약관 | > |
| 🔒 | 개인정보처리방침 | > |
| ℹ️ | 버전 정보 | 1.0.0 (1) |

### 5. 메뉴 섹션 3: 계정 (구분선 + 경고 톤)

| 아이콘 | 라벨 | 우측 |
|---|---|---|
| 🚪 | 로그아웃 | (회색) |
| 🗑 | 계정 삭제 | (빨강) |

### 6. 사진 변경 (탭 시 ActionSheet)
- "카메라로 촬영"
- "갤러리에서 선택"
- "기본 이미지로 변경"
- "취소"

### 7. 닉네임 편집 (인라인)
- 닉네임 옆 편집 아이콘 탭 → 텍스트 필드로 전환
- 우측에 "저장" / "취소" 버튼
- 1~20자 검증 (onboarding과 동일)

### 8. 차단 목록 화면 (별도 라우트 `blocked-users`)
- 차단한 사용자 리스트
- 각 항목: 아바타 + 닉네임 + "차단 해제" 버튼
- 빈 상태: "차단한 사용자가 없어요"

### 9. 계정 삭제 흐름 (별도 라우트 `account-delete`)

#### 9.1 1단계: 안내
- 풀스크린 경고 페이지
- 헤딩: **정말 계정을 삭제하시겠어요?**
- 안내:
  - 모든 영상 / 메시지 / 그룹 정보 영구 삭제
  - 복구 불가
  - **소유 그룹**: N개 → "그룹 해체 또는 권한 위임 선택 필요" (Q-12-06)
- 하단 두 버튼: "취소" / "다음"

#### 9.2 2단계: 소유 그룹 처리 (해당 시)
- 소유 그룹 리스트 표시
- 각 그룹마다 선택:
  - 그룹 해체 (모든 멤버에게 알림)
  - 권한 위임 (다른 멤버 선택)
- 모든 그룹 처리 완료 시 "다음" 활성

#### 9.3 3단계: 최종 확인
- 헤딩: "확인을 위해 '삭제'를 입력해 주세요"
- 텍스트 필드: 정확히 "삭제" 입력 시 CTA 활성
- 큰 빨간 버튼: **계정 영구 삭제**
- 삭제 진행 중 스피너 + 진행 상태 표시 (1/4 그룹 처리 중...)

## 디자인 토큰
- `profile.avatarSize` — 110
- `profile.menuItemHeight` — 56
- `profile.dangerColor` — `#E53935`
- `profile.dividerColor` — `#E0E0E0`

## 컴포넌트
- `ProfileCard` — `{ avatar, nickname, phoneMasked, onEditAvatar, onEditNickname }`
- `MenuRow` — `{ icon, label, value?, onPress, variant: 'default' | 'danger' }`
- `BlockedUserItem` — `{ user, onUnblock }`
- `AccountDeleteFlow` — 3단계 stepper
- `OwnerGroupChoice` — `{ group, onDissolve, onTransfer, members }`

## 상태별 UI
- 닉네임 편집 중 (입력 필드 + 저장/취소)
- 사진 업로드 중 (아바타 스피너)
- 로그아웃 진행 중 (전체 로딩 오버레이)
- 계정 삭제 진행 중 (단계별 진행 표시)
- 차단 해제 진행 중 (해당 항목 스피너)
