# legal — UI

## 레이아웃 (옵션 A: 외부 브라우저 호출)
이 경우 자체 화면 없음 — `expo-web-browser`로 외부 URL 열기.

## 레이아웃 (옵션 B/C: 인앱 표시)

### 1. 헤더
- 좌측 ← 뒤로가기
- 가운데: 약관 종류
  - "이용약관" (tos)
  - "개인정보처리방침" (privacy)
- 우측: 외부 브라우저 아이콘 (옵션 — 외부에서 보기)

### 2. 메타 정보 영역 (상단)
```
┌─────────────────────────────────────┐
│ 버전: v1.0                          │
│ 시행일: 2026-05-12                  │
│ 마지막 업데이트: 2026-05-12         │
└─────────────────────────────────────┘
```
- 옅은 회색 카드 또는 인라인

### 3. 본문 영역 (스크롤 가능)
- 마크다운 또는 WebView로 약관 전문 렌더링
- 목차 (옵션): 좌측 floating button → 섹션 점프
- 텍스트 검색 (옵션): 우상단 돋보기 아이콘

### 4. 마크다운 스타일링
- 헤딩 (H1/H2/H3): 굵은 sans-serif, 위계별 크기 차등
- 단락: 16px 본문, 줄간격 1.6
- 강조 (`**bold**`): 진한 그레이
- 링크: 노란색 + 밑줄
- 리스트: 들여쓰기 + 점/번호
- 표: 가로 스크롤 가능
- 인용 (`>`): 좌측 노란 보더 + 옅은 회색 배경

### 5. 푸터 (스크롤 끝)
- 작은 텍스트: "본 약관에 대한 문의는 contact@chimeme.app으로 보내주세요"
- 동의 화면에서 진입한 경우 "동의하고 돌아가기" CTA 표시

## 재동의 모달 (약관 변경 시)

### 모달 레이아웃
- 풀스크린 모달
- 닫기 버튼 없음 (강제 동의)

#### 헤딩 영역
- 큰 헤딩: **약관이 업데이트되었어요**
- 서브: "ChimeMe를 계속 이용하시려면 새 약관에 동의해 주세요"

#### 변경 항목 표시
```
변경된 약관:
☑ 이용약관 (v1 → v2)
   [내용 보기 >]
☐ 개인정보처리방침 (변경 없음)

변경 요약:
• 콘텐츠 보존 기간 명시 (7일)
• 신고 처리 절차 추가
```

#### CTA
- 큰 노란 버튼: **동의하고 계속하기** (체크 후 활성)
- 작은 텍스트 링크: "거부하시려면 계정을 삭제할 수 있습니다 [계정 삭제]"

## 디자인 토큰
| 토큰 | 값 |

| `legal.bodyFontSize` | 16 |
| `legal.lineHeight` | 1.6 |
| `legal.linkColor` | `color.primary` |
| `legal.quoteBorderColor` | `color.primary` |
| `legal.quoteBgColor` | `color.surfaceMuted` |

## 컴포넌트
- `LegalHeader` — `{ type, version, effectiveDate }`
- `LegalBody` — `{ markdown }` 또는 `{ url }` (WebView)
- `LegalUpdateModal` — `{ changes: VersionChange[], onAgree, onDelete }`
- `VersionChangeItem` — `{ type, oldVersion, newVersion, summary }`

## 상태별 UI
- 로딩 (스피너)
- 표시 (스크롤 가능)
- 외부 링크 클릭 (확인 다이얼로그 "외부 브라우저로 이동합니다")
- 재동의 모달 (체크박스 + CTA)
- 오프라인 (캐시된 약관 표시 + "마지막 업데이트: X일 전" 안내)

## 접근성

- 헤딩별 a11y heading role

- 변경 항목 a11y emphasize
