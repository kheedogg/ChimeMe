# group-settings — 그룹 관리

## 화면 키
`group-settings`

## 진입로
- `group-detail` 헤더 우측 ⋮ 메뉴
- `chat` 헤더 우측 ⋮ 메뉴

## 목적
- 그룹 정보 조회/편집 (이름, 이모지)
- 멤버 관리 (목록, 추방 - 소유자만)
- 권한 위임 (소유자 변경)
- 그룹 해체 (소유자만)
- 본인 그룹 탈퇴 (일반 멤버)
- 그룹 초대 링크 재발급

## 사용자 시나리오

### 소유자
1. 그룹 이름/이모지 편집
2. 새 초대 링크 발급
3. 멤버 추방 (선택 멤버 길게 누르기 → "그룹에서 내보내기")
4. 권한 위임 (다른 멤버 선택)
5. 그룹 해체

### 일반 멤버
1. 그룹 정보 조회 (편집 불가)
2. 멤버 목록 조회 (추방 불가)
3. 그룹 탈퇴

## 정책 (Q-12-06 확정)

| 액션 | 권한 | 처리 |
|---|---|---|
| 그룹 이름/이모지 편집 | 소유자만 | 즉시 반영, 멤버에 시스템 메시지 |
| 멤버 추방 | 소유자만 | 추방자 영상 → 채팅 placeholder, FCM 구독 해제, 차단 옵션 |
| 권한 위임 | 소유자만 | 새 소유자 지정 후 자동 위임, 시스템 메시지 |
| 그룹 해체 | 소유자만 | 모든 영상/채팅/멤버십 일괄 삭제, 멤버에 FCM 알림 |
| 본인 탈퇴 | 모든 멤버 | 본인 영상 잔존(작성자 "(탈퇴한 사용자)"로 익명화) |

## 소유자 탈퇴 특수 처리
- 소유자가 탈퇴 시도 시: **그룹 해체 강제 선택 UI** (Q-12-06)
- 옵션: "그룹 해체" / "다른 멤버에게 권한 위임 후 탈퇴" / "취소"
- 권한 위임 후 자동 탈퇴 진행

## 미정 / 향후
- 관리자(admin) 역할 추가 — 현재는 owner / member 2단계
- 그룹 공개/비공개 설정 — 현재 모든 그룹은 초대제 비공개
