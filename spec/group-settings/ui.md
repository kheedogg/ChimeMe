# group-settings — UI

## 레이아웃

### 1. 헤더
- 좌측 ← 뒤로가기
- 가운데: **그룹 설정**

### 2. 그룹 정보 카드 (상단)
```
       ┌──────┐
       │ 😀   │  ← 탭 시 이모지 변경 (소유자만)
       └──────┘
        [가족 ✏]      ← 편집 아이콘 (소유자만)
       3명 · 2026-05-01에 생성됨
```
- 이모지 110×110 카드
- 그룹 이름 + 편집 아이콘 (소유자만 노출)
- 멤버 수 + 생성일

### 3. 초대 링크 섹션 (소유자/멤버 모두)
```
┌──────────────────────────────────────┐
│ 초대 링크                            │
│ https://chimeme.app/invite/Ab3xZ9    │
│  [복사]  [새로 만들기*]             │
│ 🕒 6일 23시간 남음                   │
└──────────────────────────────────────┘
* 소유자만
```
- "새로 만들기": 기존 링크 만료시키고 신규 발급 (소유자만)
- 만료까지 남은 시간 카운트다운

### 4. 멤버 섹션
```
멤버 (3명)
─────────────────────────
[아바타] 도희 (소유자)        [...]
[아바타] 민지                  [...]
[아바타] 진우                  [...]
```
- 본인은 "(나)" 표시
- 소유자는 "(소유자)" 칩 표시
- 우측 ⋮ 메뉴 (소유자가 다른 멤버 항목에서 탭 시):
  - 프로필 보기
  - 신고하기
  - **그룹에서 내보내기** (소유자만)

### 5. 본인 액션 섹션 (구분선)

#### 일반 멤버
```
[그룹 나가기]   ← 노란/빨간 톤
```

#### 소유자
```
[권한 위임하기]
[그룹 해체]    ← 빨강 강조
```

### 6. 그룹 나가기 다이얼로그 (일반 멤버)
- 헤딩: "정말 그룹을 떠나시겠어요?"
- 안내:
  - 내가 올린 영상은 그룹에 남으며 작성자는 "(탈퇴한 사용자)"로 표시됨
  - 채팅 메시지도 동일 처리
  - 재가입 시 새 초대 링크 필요
- 버튼: "취소" / "그룹 나가기" (빨강)

### 7. 멤버 추방 다이얼로그 (소유자)
- 헤딩: "[닉네임]을 그룹에서 내보내시겠어요?"
- 옵션 체크박스: "이 사용자가 다시 가입하지 못하게 차단"
- 안내: "내보낸 사용자의 영상은 그룹에 남으며, 채팅 메시지는 '(강퇴된 사용자)'로 표시됩니다"
- 버튼: "취소" / "내보내기" (빨강)

### 8. 권한 위임 흐름 (소유자)

#### 8.1 멤버 선택
- 헤딩: "누구에게 권한을 위임할까요?"
- 멤버 리스트 (본인 제외)
- 각 멤버 선택 시 라디오 표시

#### 8.2 확인
- 헤딩: "[닉네임]에게 권한을 위임하시겠어요?"
- 안내: "위임 후 [닉네임]이 그룹 관리 권한을 가지며, 본인은 일반 멤버가 됩니다"
- 버튼: "취소" / "위임하기"

### 9. 그룹 해체 흐름 (소유자)

#### 9.1 1단계 안내
- 풀스크린 경고
- 헤딩: **정말 그룹을 해체하시겠어요?**
- 안내:
  - 모든 영상 / 채팅 / 멤버십 영구 삭제
  - 복구 불가
  - 모든 멤버에게 알림 발송
- 버튼: "취소" / "다음"

#### 9.2 2단계 확인
- 헤딩: "확인을 위해 그룹 이름을 입력해 주세요"
- 텍스트 필드: 그룹 이름 정확히 입력 시 CTA 활성
- 큰 빨간 버튼: **그룹 영구 해체**

## 컴포넌트
- `GroupInfoCard` — `{ group, isOwner, onEditName, onEditEmoji }`
- `InviteLinkSection` — `{ invite, isOwner, onCopy, onRefresh }`
- `MemberListItem` — `{ member, isOwner, isMe, onKick, onReport }`
- `LeaveGroupDialog` — `{ onConfirm, onCancel }`
- `KickMemberDialog` — `{ member, onConfirm, onCancel, blockOption }`
- `TransferOwnershipFlow` — 2단계 stepper
- `DissolveGroupFlow` — 2단계 stepper

## 상태별 UI
- 일반 멤버 (편집 아이콘 / 추방 메뉴 미노출)
- 소유자 (모든 액션 노출)
- 편집 중 (이름 입력 + 저장/취소)
- 추방 처리 중 (스피너)
- 권한 위임 처리 중
- 그룹 해체 처리 중 (진행 상태 표시)
