# group-detail — Interactions

## 진입
- params: `groupId`, (옵션) `slot` ISO 시간 — 없으면 현재 시간으로
- 첫 진입 시 unread 카운트 0으로 갱신

## 시간대 이동
- 트리거: 좌/우 화살표 탭, 본문 좌우 스와이프, 시간 네비 영역 좌우 스와이프
- 경계:
  - 그룹 생성 시간 이전으로는 이동 불가 → ← disabled
  - 미래 시간대로 이동 불가 → → disabled (현재 슬롯이 최신)
- 시간대 변경 시 모든 영상 일시정지 → 새 슬롯의 영상 자동 재생(음소거)

## 영상 재생
- 자동 재생: ON
- 사운드 기본: OFF (음소거)
- 첫 탭 → 사운드 ON, "탭하여 소리 켜기" 토스트 페이드아웃
- 이후 탭 → 사운드 토글 (음소거 ↔ 켜기)
- 화면을 떠나면 모든 영상 일시정지/언로드

## 미업로드 슬롯
- 다른 멤버: 회색 placeholder만 표시
- 본인 + **현재 슬롯(업로드 윈도우 60분 내)**: "지금 촬영" CTA → `capture/[groupId]`로 이동
- 본인 + 과거 슬롯: 그냥 placeholder (소급 업로드 금지 — 슬롯 마감 후 잠금)

## 페이지 인디케이터
- 그룹이 가진 슬롯 개수만큼 dot 표시
- 활성 dot = 현재 보고 있는 슬롯
- dot 자체는 보통 read-only(인터랙션 가능성 향후 검토)

## 도움말 모달
- ? 버튼 탭 → 사용법 모달 (간단 일러스트 + 텍스트)

## 채팅 진입 (향후)
- 헤더 우측 채팅 아이콘 또는 별도 탭 진입점
- 현재 그룹의 채팅(`chat/[groupId]`)으로 이동

## 에러
- 영상 로드 실패 → 인라인 안내 + 재시도
- 그룹 접근 권한 없음 → 메인으로 되돌리고 토스트

## 접근성
- 시간 네비: "현재 09시, 이전/다음 시간대로 이동"
- 영상 카드: "<멤버명>의 <시각> 영상, 자동 재생 중, 탭하여 소리 켜기"
- 미업로드 슬롯: "<멤버명>은 이 시간대에 영상을 올리지 않았어요"
