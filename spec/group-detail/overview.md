# group-detail — 그룹 상세 (시간대별 영상)

## 화면 키
`group-detail`

## 진입로
- 메인의 그룹 카드 탭
- 푸시 알림 deeplink (특정 그룹의 특정 시간 슬롯)
- 그룹 채팅 화면의 헤더 탭 (옵션, 향후)

## 목적
- 같은 그룹 멤버들이 같은 시간대에 올린 3초 영상을 한 묶음으로 시청
- 시간대를 좌우 스와이프로 이동하며 그룹의 하루를 회고/탐색
- 미업로드 멤버에 대한 사회적 압력(부드럽게) — "아직 영상이 없어요" 안내

## 사용자 시나리오
1. **지금 보기**: 진입 시 현재 시간 슬롯("지금") 활성. 멤버별 영상 자동 재생.
2. **과거 둘러보기**: ← 화살표 또는 좌측 스와이프로 이전 시간대 이동. 미업로드 멤버는 회색 placeholder.
3. **사운드 듣기**: 음소거 기본 → 영상 탭 → 사운드 ON 토스트 사라짐.
4. **내가 안 올린 슬롯**: 본인 슬롯에 영상 없으면 "지금 촬영" CTA (현재/미래 슬롯에서만, 향후 확장).

## 성공 지표
- 시간대 간 전환 응답성 (자동 재생 끊김 없음)
- 영상 자동 재생 음소거 정책으로 "탭하여 소리 켜기" 인터랙션이 자연스러울 것
