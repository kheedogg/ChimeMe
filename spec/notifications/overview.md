# notifications — 알림 설정

## 화면 키
`notifications`

## 진입로
- 프로필 → 알림

## 목적
- 푸시 알림 종류별 ON/OFF 및 시간대 설정

## 사용자 시나리오
1. 영상 알림 (매시 정각 트리거) ON/OFF
2. 채팅 알림 ON/OFF
3. 방해 금지 시간 설정 (예: 23:00 ~ 07:00 알림 차단)
