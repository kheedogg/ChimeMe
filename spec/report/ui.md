# report — UI

## 표시 형태
신고 흐름은 **모달 바텀시트**로 표시 (별도 라우트가 아닌 오버레이). 진입 트리거 화면 위에 표시.

## 레이아웃

### 1. 헤더
- 좌측: ← 닫기 (X 아이콘)
- 가운데: **신고하기**

### 2. 신고 대상 정보 (최상단 카드)
```
┌─────────────────────────────────────┐
│ [썸네일/아바타]                      │
│                                      │
│ 영상 (또는 메시지 / 사용자)          │
│ 작성자: 민지                         │
│ 시각: 2026-05-12 14:23               │
└─────────────────────────────────────┘
```
- 대상 유형 표시 (영상/메시지/사용자)
- 영상: 썸네일 + 작성자 + 슬롯
- 메시지: 첫 줄 미리보기 + 작성자 + 시각
- 사용자: 아바타 + 닉네임

### 3. 사유 선택 섹션
**라디오 그룹** (단일 선택):
- ⚪ 욕설 / 혐오 표현
- ⚪ 음란물 / 부적절한 콘텐츠
- ⚪ 스팸 / 광고
- ⚪ 폭력 / 위협
- ⚪ 미성년자 보호 위반
- ⚪ 기타

각 사유 옆 작은 도움말 아이콘 (탭 시 상세 설명 툴팁).

### 4. 부연 설명 (선택)
```
┌─────────────────────────────────────┐
│ 추가 설명 (선택)                     │
│                                      │
│ [텍스트 영역, 최대 500자]            │
│                                      │
└─────────────────────────────────────┘
                              0 / 500
```
- "기타" 선택 시에는 필수 입력
- 글자 수 카운터 우측 하단

### 5. 안내 영역
- 작은 텍스트: "신고는 익명으로 처리됩니다. 동일 콘텐츠에 3건 이상 신고가 누적되면 그룹 전체에서 가려집니다"

### 6. CTA (모달 하단 고정)
- 큰 빨간 버튼: **신고하기**
- 비활성 조건:
  - 사유 미선택
  - "기타" 선택 시 부연 설명 1자 미만

## 신고 완료 화면 (제출 후)
- 모달 본문 교체:
  - 큰 체크 아이콘 (녹색)
  - "신고가 접수됐어요"
  - 서브: "검토 후 부적절한 콘텐츠는 그룹 전체에서 가려집니다"
- CTA: **확인** (모달 닫기)

## 신고 횟수 초과 화면 (24시간 제한)
- 모달 본문:
  - 경고 아이콘
  - "신고 기능을 일시적으로 사용할 수 없어요"
  - 서브: "허위 신고 누적으로 24시간 동안 신고 기능이 제한됩니다"
  - 남은 시간 카운트다운
- CTA: **확인**

- 기본 (사유 미선택, CTA 비활성)
