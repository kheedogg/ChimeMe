# chat — 그룹 채팅

## 화면 키
`chat`

## 진입로
- 그룹 상세 헤더의 채팅 아이콘 (UI 추가 필요)
- 메인 그룹 카드에서 별도 진입로 추가 검토

## 목적
- 같은 그룹 멤버들과 텍스트/간단 첨부로 대화
- **그룹 채팅만** 지원, DM 없음 (확정)

## 사용자 시나리오
1. 그룹 채팅방 진입
2. 메시지 입력/전송, 멤버 메시지 실시간 수신
3. (옵션) 사진/이모지/이전 영상 인용

## 정책 (확정 2026-05-11)
- **메시지 형식**: 텍스트 + 이모지 + **그룹의 이전 영상 인용** (외부 사진/파일 첨부 없음)
  - 영상 인용은 `groupId/postId` 참조로 저장. 채팅 뷰에서 썸네일 + 시간 슬롯 메타로 렌더
- **읽음 표시**: "모두 읽음" 만 표시. 멤버 전원이 읽으면 인디케이터 사라짐. 개별 멤버 read status는 노출 안 함
- **삭제/수정**: 본인 메시지 **삭제만 허용, 수정 불가**. 삭제된 메시지는 "삭제된 메시지" placeholder로 자리 유지
- **알림**: 영상 알림과 **분리**된 별도 ON/OFF 토글 (`notifications` 화면 참고)

## 확정 정책 (2026-05-12)
- **페이지네이션 단위**: **50건/페이지** (D-12-08)
- **본인 메시지 삭제 시간 제한**: **5분** (Q-12-05) — Security Rule에서 시간 기반 검증
- **메시지 정렬**: `serverTimestamp` 기반 (D-12-08, 클라이언트 시계 무관)
- **멱등성 키**: `clientMessageId = uuid()` → Firestore 문서 ID로 사용 → 재전송 시 덮어쓰기로 중복 방지
- **영상 인용 카드 탭**: 그룹 상세의 해당 슬롯으로 점프 + 인용된 영상 자동 재생
- **신고 정책 (Q-12-04)**:
  - 신고 즉시 신고자 화면에서 가림
  - 누적 3건 신고 시 그룹 전체 가림 + 관리자 검토
  - 허위 신고 5건 누적 시 신고 기능 24시간 정지
- **차단 정책**: 차단 시 기존 메시지도 클라이언트에서 즉시 가림
- **채팅 보존 기간**: **90일** (Q-12-08) — Cloud Scheduler 매일 03:00 UTC 정리
