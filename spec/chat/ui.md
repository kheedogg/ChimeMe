# chat — UI

## 레이아웃

### 1. 헤더
- 좌측 ← 뒤로가기 (그룹 상세로)
- 가운데:
  - 그룹명 (굵게)
  - 서브: "N명 · 모두 읽음" 또는 "N명 · 안 읽은 멤버 N명"
- 우측: 그룹 설정 아이콘 (⋮) — 향후 `group-settings`로 연결

### 2. 메시지 리스트 (스크롤)
- 최신 메시지가 하단, 위로 스크롤 시 과거 메시지 페이지네이션 (50건씩 — D-12-08)
- 메시지 아이템:

#### 본인 메시지 (우측 정렬)
```
                      [내가 보낸 메시지]
                              09:42 · 모두 읽음
```
- 배경: 노란 톤 (`color.primary`의 옅은 변형)
- 둥근 모서리 (좌측 모서리 더 크게)
- 시간 + 읽음 인디케이터 (메시지 아래 우측)

#### 다른 멤버 메시지 (좌측 정렬)
```
[아바타]
       [멤버명]
       [메시지 내용]
       09:43
```
- 아바타 32×32 원형
- 멤버명: 작은 칩 또는 텍스트
- 메시지 배경: 회색 톤
- 연속 메시지(동일 발신자)는 아바타/이름 생략, 시간만 마지막에 표시

#### 영상 인용 메시지 (확정 정책)
```
┌──────────────────┐
│ [썸네일]         │
│ 🕐 09:00 슬롯    │  ← 탭 시 그룹 상세의 해당 슬롯으로 점프
│ 📹 작성자: 민지   │
└──────────────────┘
```
- 카드 형태, 라운드 12
- 썸네일 + 슬롯 메타 + 작성자

#### 삭제된 메시지
```
[삭제된 메시지]
09:43
```
- 회색 이탤릭, 어색하지 않게 자리 유지

### 3. 시스템 메시지 (가운데 정렬)
- "민지님이 그룹에 참여했어요" (옅은 회색, 작은 텍스트)
- "민지님이 그룹을 떠났어요"
- "민지님이 강퇴되었어요"
- "운영자에 의해 일부 메시지가 가려졌어요" (Q-12-04 신고 정책)

### 4. 입력 영역 (하단 고정)
- 좌측 + 버튼 → ActionSheet:
  - "이전 영상 인용"
  - "이모지 추가"
- 가운데: 텍스트 입력 (멀티라인, 자동 높이 조절, 최대 4줄)
- 우측: 전송 버튼 (텍스트 비어있을 때 비활성)

### 5. 빈 상태
- 메시지 0건 → 가운데 일러스트 + "아직 메시지가 없어요. 첫 인사를 남겨보세요"

### 6. 신고 / 메뉴 (메시지 길게 누르기)
- 본인 메시지: "삭제" (5분 이내만 활성 — Q-12-05)
- 타인 메시지: "신고하기" / "사용자 차단" / "복사"

## 디자인 토큰
- `chat.bubbleMaxWidth` — 70% of screen
- `chat.myBg` — `color.primary` 옅은 변형
- `chat.otherBg` — `color.surfaceMuted`
- `chat.radius` — 16 (꼬리 모서리 8)
- `chat.systemTextColor` — 옅은 회색

## 컴포넌트
- `ChatHeader` — `{ groupName, memberCount, allRead }`
- `MessageList` — 가상화된 인버티드 FlatList
- `MessageBubble` — `{ message, isMine, prevMessage, isLast }`
- `VideoQuoteCard` — `{ groupId, postId, slotKey, authorName, thumbnail }`
- `SystemMessage` — `{ text, type }`
- `MessageInput` — `{ value, onChange, onSend, onAttach }`
- `MessageActionSheet` — 길게 누르기 시 표시 (삭제/신고/차단)

## 상태별 UI
- 메시지 발송 중 (반투명 + 스피너)
- 발송 실패 (빨간 인디케이터 + 재시도 버튼)
- 5분 경과 (삭제 버튼 비활성)
- 모두 읽음 (헤더 인디케이터 변화)
- 페이지네이션 로딩 (상단 스피너)

## 접근성
- 메시지 a11y: "민지가 09:43에 보낸 메시지: [내용]"
- 영상 인용 a11y: "민지의 09:00 슬롯 영상 인용, 탭하면 영상으로 이동"
- 입력 영역 placeholder a11y label
- 길게 누르기 메뉴 a11y label

## 키보드 처리
- 입력 포커스 시 메시지 리스트 자동 스크롤 (최신 메시지로)
- iOS: `KeyboardAvoidingView` (`padding`)
- Android: `windowSoftInputMode="adjustResize"`

## 영상 인용 흐름 (UI)
1. + 버튼 → "이전 영상 인용" → 그룹 상세의 슬롯 picker 모달
2. 슬롯 선택 → 본인의 영상이 있으면 선택, 없으면 다른 멤버 영상도 선택 가능 (그룹 내 공개)
3. 인용 카드 input 영역 위에 미리보기 → 텍스트 추가 가능
4. 전송 시 메시지 + 인용 함께 발송
