# auth-verify — SMS 코드 확인

## 화면 키
`auth-verify`

## 진입로
- `auth-phone`에서 코드 발송 성공 시

## 목적
- 사용자가 받은 6자리 인증 코드를 입력하여 로그인 완성

## 사용자 시나리오
1. 6자리 코드 입력 (자동 채움 지원 — iOS/Android SMS auto-fill)
2. 자동 검증 또는 "확인" 탭 → 성공 시 메인으로
3. 신규 사용자면 `users/{uid}` 문서 생성 + (옵션) 프로필 입력 화면
