# group-create — UI

## 레이아웃 (모달 바텀시트, 65% 높이)

### 1. 상단
- 좌측 ← 닫기 (X 아이콘)
- 가운데: **새 그룹 만들기**

### 2. 그룹 정보 영역

#### 2.1 그룹 이모지 (선택)
```
       ┌────┐
       │ 😀 │   ← 탭 시 이모지 picker 모달
       └────┘
```
- 기본: 랜덤 이모지 (😀 🎉 🌟 등)
- 탭 시 이모지 picker 표시 (인기 카테고리 + 검색)

#### 2.2 그룹 이름 입력
- 라벨: "그룹 이름"
- 1~30자 제한
- 헬퍼 텍스트: "예: 우리 가족, 동아리 친구들"
- 글자 수 카운트 (예: `5 / 30`)

### 3. 초대 방법 (Q-12-10 — SMS 딥링크 + QR)

#### 3.1 초대 링크 영역 (그룹 생성 직후 자동 발급)
```
┌──────────────────────────────────────┐
│ https://chimeme.app/invite/Ab3xZ9    │
│                              [복사]  │
└──────────────────────────────────────┘
🕒 링크는 7일 동안 유효해요
```

#### 3.2 공유 액션 버튼들
- 가로 스크롤 또는 그리드:
  - 📱 **SMS로 보내기** (탭 → 연락처 선택 시트)
  - 📷 **QR 코드 보기** (탭 → 풀스크린 QR 모달)
  - 📋 **링크 복사**
  - 🟡 **카톡 공유** (P2 — 베타 비활성)

#### 3.3 멤버 카운터
- "**1명** / 최대 8명" — 본인 1명 + 가입자 카운트

### 4. CTA (모달 하단 고정)
- 큰 노란 버튼: **그룹 만들기** (이름 1자 이상 시 활성)
- 생성 후 바로 그룹 상세로 이동

### 5. 보조 영역
- 작은 텍스트 (CTA 아래): "그룹은 최대 8명까지 가입할 수 있어요"

## 디자인 토큰
- `modal.height` — 65% (확장 가능)
- `radius.modalTop` — 24
- `invite.linkBg` — `color.surfaceMuted`

## 컴포넌트
- `GroupNameInput` — `{ value, onChange, max=30 }`
- `EmojiPicker` — `{ value, onChange }`
- `InviteLinkBox` — `{ url, onCopy, expiresAt }`
- `ShareActionRow` — `{ actions: [...], onAction }`
- `MemberCounter` — `{ current, max }`
- `QRCodeModal` — `{ url, onClose }`

## 상태별 UI
- 기본 (이름 빈 값, CTA 비활성)
- 입력 중 (글자 수 업데이트)
- 생성 중 (CTA 스피너 + 입력 잠금)
- 생성 완료 (그룹 상세로 자동 이동, 모달 자동 닫힘)
- 생성 실패 (인라인 에러 + CTA 활성 복귀)
- QR 모달 표시 중
- SMS 공유 시 시스템 share sheet 호출

## 빈 상태 / 에러
- 그룹 이름 부적절 (욕설) → 인라인 에러
- 그룹 이름 중복 — **허용** (사용자가 직접 식별 가능, 시스템 제약 없음)
- 네트워크 오류 → 토스트

## 접근성
- 이모지 picker a11y label: "그룹 이모지 선택"
- 초대 링크 복사 a11y: "초대 링크가 클립보드에 복사되었습니다"
- QR 코드 a11y description: "그룹 초대 QR 코드, 카메라로 스캔하면 가입 화면으로 이동"

## 향후 확장
- 그룹 설명 필드 (선택 입력)
- 멤버 추가 후 그룹 생성 (현재는 본인만 자동 가입)
- 그룹 공개/비공개 설정 (현재 모든 그룹은 초대제로 비공개)
