# group-create — 그룹 생성 · 초대

## 화면 키
`group-create`

## 진입로
- 메인 우측 + FAB

## 목적
- 신규 그룹 생성 + 멤버 초대

## 사용자 시나리오
1. 그룹 이름 입력
2. 초대 방법 선택:
   - 연락처에서 선택 (권한 필요)
   - 초대 링크 공유 (deeplink)
3. 생성 완료 → 새 그룹의 그룹 상세로 이동

## 정책
- **그룹 최대 인원: 8명** (확정 2026-05-11)
